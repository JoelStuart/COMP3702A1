/*************COMPILATION & RUN GUIDE ***************\

	/*To compile*\

    To compile, run from this folder in cmd (folder with src and structure in it):

    javac -cp src src/com/company/NavApp.java

    ** Location: containing folder **


	/*To run*\

    To run, run this from within folder src:

    java com.company.NavApp {args}

    **Location: containingfolder/src **
	(this means input files need to be in src folder)
